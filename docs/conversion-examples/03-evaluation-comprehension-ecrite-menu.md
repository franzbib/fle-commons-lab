# Comprendre un menu de restaurant

## Résumé

Évaluation courte A2 pour lire un menu simple, repérer des prix et choisir des éléments compatibles avec un budget.

## Objectifs

- Identifier des catégories dans un menu.
- Repérer des prix.
- Répondre à des questions simples sur un document écrit.
- Justifier un choix à partir d’une contrainte de budget.

## Déroulé

1. Distribuer le document et les questions.
2. Laisser deux minutes de lecture silencieuse.
3. Faire répondre individuellement aux questions.
4. Corriger collectivement ou ramasser selon l’objectif d’évaluation.
5. Reprendre le lexique utile: entrée, plat, dessert, coûte, moins cher.

## Support Étudiant

### Document

Menu d’un restaurant:

| Catégorie | Choix | Prix |
| --- | --- | --- |
| Entrée | soupe de légumes | 6€ |
| Entrée | salade verte | 5€ |
| Plat | poulet rôti | 14€ |
| Plat | poisson du jour | 16€ |
| Plat | omelette | 11€ |
| Dessert | tarte aux pommes | 6€ |
| Dessert | glace | 5€ |

### Questions

1. Quel est le plat le moins cher ?
2. Combien coûte le poisson du jour ?
3. Vous avez 20€. Choisissez une entrée et un plat.
4. Le menu propose-t-il un dessert aux fruits ?

## Guide Professeur

- Vérifier que les apprenants comprennent la différence entre entrée, plat et dessert.
- Ne pas expliquer tout le vocabulaire avant la lecture si l’objectif est évaluatif.
- Accepter plusieurs réponses possibles à la question 3 si le budget est respecté.
- Ajouter un barème si l’évaluation doit être notée.

## Corrigé

1. Le plat le moins cher est l’omelette.
2. Le poisson du jour coûte 16€.
3. Réponses possibles:
   - salade verte + omelette = 16€;
   - soupe de légumes + omelette = 17€;
   - salade verte + poulet rôti = 19€;
   - soupe de légumes + poulet rôti = 20€.
4. Oui, il y a la tarte aux pommes.

## Variantes

- Ajouter une boisson et demander de calculer un repas complet.
- Transformer l’évaluation en jeu de rôle serveur-client.
- Pour un niveau A1, réduire le nombre de plats et poser uniquement des questions de repérage.
- Pour un niveau B1, demander une justification écrite du choix.

## Ce Qui Est Perdu Dans La Conversion

- Apparence visuelle d’un vrai menu.
- Images, colonnes, typographie ou indices graphiques.
- Authenticité éventuelle du document source.
- Informations de droits si le menu vient d’un restaurant réel.

## Automatisation Possible

- Convertir les questions numérotées en Markdown.
- Séparer le corrigé quand il est explicitement annoncé.
- Transformer une liste de prix en tableau Markdown si la structure est régulière.
- Proposer le type `évaluation` et la compétence `compréhension écrite`.

## Relecture Enseignant Nécessaire

- Vérifier toutes les réponses possibles pour la question de budget.
- Confirmer le niveau A2.
- Ajouter un barème si nécessaire.
- Vérifier les droits si le document source est authentique.
