# Jeu du Monstre : Les parties du corps

## Objectifs
- Lexique : les parties du corps, les nombres, les couleurs.
- Action : Dessiner sous la dictée, puis dicter.

## Déroulé
1. Le professeur dicte au tableau (5 min).
2. Les étudiants dessinent en binômes (15 min).

## Support étudiant
*(Pas de support écrit : l'activité est purement orale)*

## Guide professeur
### Étape 1 : Le professeur dicte
Allez au tableau. Demandez aux étudiants de vous dicter un monstre : "Dessinez trois yeux rouges". Dessinez exactement ce qu'ils disent.

### Étape 2 : Les étudiants dessinent
En binômes. L'étudiant A dessine un monstre en cachette. Il le dicte à l'étudiant B. 
On inverse les rôles.

## Corrigé
Activité d'expression libre, pas de corrigé type. Le dessin valide la bonne compréhension.
